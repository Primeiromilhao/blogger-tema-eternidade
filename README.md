# Tema Blogger - Eternidade, Tempo e a Cruz

## Descrição

Tema completo para Blogger com design responsivo e moderno para o estudo "Eternidade, Tempo e a Cruz". Desenvolvido com HTML, CSS e JavaScript usando Tailwind CSS para estilos.

## Características

- ✨ Design responsivo e moderno
- 🎨 Tailwind CSS para estilos
- 📱 Compatível com dispositivos móveis
- 🎯 Otimizado para Blogger
- 🔄 Estrutura XML para temas Blogger

## Arquivo Principal

- `index.html` - Tema Blogger completo com código XML e HTML

## Como Usar

1. Faça login no seu painel do Blogger
2. Vá para **Tema** > **Editar HTML**
3. Copie todo o conteúdo de `index.html`
4. Cole no editor de HTML do Blogger
5. Clique em **Salvar tema**

## Conteúdo do Estudo

O tema aborda:
- Visão Geral sobre Eternidade e Tempo
- Chronos vs Kairos
- João e o Apocalipse
- Abraão e a Cidade Celestial
- A Cruz na Eternidade

## Customização

Você pode customizar as cores alterando os valores hexadecimais:
- `#0f172a` - Azul escuro (soberBlue)
- `#881337` - Vinho (soberWine)
